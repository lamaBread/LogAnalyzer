# 필요한 주요 명령어

1. ```sudo docker logs -f php-apache```
2. ```node apiScheduler.js```
3. ```docker exec -it {container_name} /bin/bash```
4. ```sudo docker compose up -d --build```
5. ```sudo docker compose down```
6. ```sudo docker compose stop {container_name} && sudo docker compose rm -f {container_name}``` (정확하지 않을 수 있음.)


# 주요 URL

1. 경고 페이지: ```http://localhost:8445/APIs/alertDisplayPage/index.php```
2. 메인 페이지: ```http://localhost:8445/```
3. 공격 대상 페이지: ```http://localhost/```
